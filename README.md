Freelance Portfolio & Chat Platform

A full-stack, real-time Freelance Portfolio and Chat platform built with the MERN stack (MongoDB, Express, React, Node.js) and Socket.io. This platform allows freelancers to create aesthetically pleasing portfolio profiles, display their skills and services, and communicate with clients seamlessly in real-time.

 🚀 Features

**Freelancer Profiles**
- **Dynamic Portfolios:** Create and manage customized portfolio profiles.
- **Template Engine:** Built-in portfolio templates (e.g., Corporate Classic) to beautifully present work and experience.
- **Dashboard & Profile Management:** Fully featured dashboard for freelancers to update their skills, bio, and past projects.

 **Real-Time Communication**
- **Instant Messaging:** Real-time chat powered by Socket.io.
- **Chat Inbox:** Two-panel chat dashboard to easily manage all ongoing client conversations.
- **Live Notifications:** Unread message badges and instant alerts for new messages.

 **Authentication & Security**
- **User Authentication:** Secure user registration and login.
- **Password Encryption:** Passwords securely hashed using bcrypt.
- **JWT Authorization:** Token-based authentication for securing API routes.

**Frontend:**

**Backend:**
